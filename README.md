## How to run
python save_object.py input.json

## How to run tests
python test.py

## Project structure
- save_object.py — main script
- get_json.py — reads JSON file
- structure_control.py — validates fields
- add_attributes.py — adds system fields
- save_json.py — saves to disk with versioning
- versions.py — shows diff between versions
- test.py — test scenario
